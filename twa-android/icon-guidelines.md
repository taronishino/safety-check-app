# アイコンガイドライン

## 必要なアイコンサイズ

### アプリアイコン（必須）
- 512 x 512px (PNG, 32-bit, アルファチャンネル付き)
- ファイル名: icon-512.png

### Play Store用グラフィック

#### 1. フィーチャーグラフィック
- 1024 x 500px (PNG or JPEG)
- ストアのトップに表示される
- ファイル名: feature-graphic.png

#### 2. スクリーンショット（最低2枚、最大8枚）
- スマートフォン: 最小 320px、最大 3840px
- 推奨: 1080 x 1920px (16:9)
- ファイル名: screenshot-1.png, screenshot-2.png...

#### 3. 高解像度アイコン
- 512 x 512px (PNG, 32-bit)
- Play Store表示用
- ファイル名: hi-res-icon.png

## アイコンデザイン案

### メインアイコン
```
背景: グラデーション (#4ECDC4 → #44A5A3)
前景: 白いハートの中に2つのシルエット
      - 大きいシルエット（親）
      - 小さいシルエット（子）
      - 点線でつながっている
```

### カラーパレット
- プライマリ: #4ECDC4 (ターコイズ)
- セカンダリ: #FF6B6B (コーラルレッド)
- 背景: #FFFFFF
- テキスト: #333333

## スクリーンショット構成案

1. **ホーム画面**
   - 役割選択画面
   - 「親として使う」「子として使う」ボタン

2. **親アプリ - メイン画面**
   - 最終操作時間の表示
   - 安否報告ボタン
   - シンプルで見やすいUI

3. **子アプリ - 親の状態確認**
   - 緑色の正常状態
   - 「最終操作から3時間25分」の表示

4. **警告状態の表示**
   - 黄色、オレンジ、赤の各状態
   - わかりやすい色分け

5. **プレミアム機能**
   - 複数親の同時見守り
   - 位置情報確認画面

## 作成ツール

### 無料ツール
1. **Canva** (https://www.canva.com)
   - テンプレートあり
   - 簡単操作

2. **Figma** (https://www.figma.com)
   - プロ仕様
   - 共同作業可能

3. **GIMP** (https://www.gimp.org)
   - 高機能
   - 完全無料

### アイコン作成のヒント
- シンプルで識別しやすいデザイン
- 小さいサイズでも認識可能
- 背景との十分なコントラスト
- 角丸は自動適用されるので四角で作成
